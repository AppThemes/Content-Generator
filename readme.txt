== Description ==

Allows you to generate content on your site randomly.

= Supports =
* ClassiPress
* Clipper
* Quality Control
* Taskerr
* Vantage


== Installation ==

1. Upload the folder 'content-generator' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to "Tools->Content Generator" menu and provide repository data.
